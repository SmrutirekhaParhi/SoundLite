## Component Specifications

Arduino Nano:
- Operating Voltage: 5V
- Input Voltage: 7-12V
- Digital I/O: 14 pins
- Analog Inputs: 8 pins
- Clock Speed: 16 MHz
- Flash Memory: 32 KB
- SRAM: 2 KB

Sound Sensor Module:
- Operating Voltage: 3.3V - 5V
- Digital Output: HIGH when sound detected
- Frequency Response: 100 Hz - 10 kHz
- Sensitivity: Adjustable via potentiometer
- Response Time: <10ms

RGB LED:
- Type: Common Cathode
- Red Forward Voltage: 2.0V
- Green Forward Voltage: 2.2V
- Blue Forward Voltage: 3.2V
- Max Current per Color: 20mA
- Brightness: ~5000 mcd

220 Ohm Resistor:
- Resistance: 220 Ω ±5%
- Power Rating: 1/4W
- Purpose: Current limiting for LEDs

## Notes

- Resistor values can be measured with a multimeter if unsure
- Source components locally if possible to avoid shipping delays
- Genuine Arduino boards recommended for reliability
- Check sensor specifications before purchase
